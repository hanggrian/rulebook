# Documentation Tag Description

Description of certain tags is a sentence, therefore, must end with `.`, `?` or `!`.

## Affected Tags

| Tag | Checkstyle | Ktlint |
| --- | :---: | :---: |
| @param | &check; | &check; |
| @return | &check; | &check; |
| @constructor | &cross; | &check; |
| @property | &cross; | &check; |
| @receiver | &cross; | &check; |
