Codestyle

Personal Java/Kotlin code convention and rules.

Download

Ktlint

Integrate ktlint to Gradle project. Using configuration ktlint, add this project as dependency. 

repositories {
    mavenCentral()
}

configurations {
    ktlint
}

dependencies {
    ktlint "com.hendraanggrian.codestyle:ktlint:$version"
}

Checkstyle

Copy the directory of config in this repository. Apply Checkstyle Plugin. Using configuration checkstyle, add this project as dependency. 

plugins {
    checkstyle
}

dependencies {
    checkstyle "com.hendraanggrian.codestyle:checkstyle:$version"
}
